# Disclaimer

## What This Is

Cortex-Agentic is a private multi-agent orchestration system built and operated by Markus Beermann for personal use.

It coordinates specialized AI agents across private projects: planning, building, reviewing, monitoring. It is not a product, not a framework, and not intended for third-party deployment.

## What This Is Not

- Not a finished product
- Not a supported open-source library
- Not a general-purpose agent framework you can drop into your project
- Not a claim of full autonomous operation

The system is in active development. Interfaces, contracts, and behaviors change without notice.

## Use of AI

This system uses the Anthropic Claude API for agent execution. Generated outputs are reviewed and controlled by a human operator. Autonomous operation is intentionally limited — human approval gates exist throughout the pipeline.

Code in this repository is partly written by AI coding agents (Codex) under human supervision and direction.
